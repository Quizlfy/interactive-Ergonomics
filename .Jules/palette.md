## 2024-05-14 - Skip Links in Single Page Sites
**Learning:** Adding a "skip to main content" link is especially important for single-page sites with heavy navigation, allowing keyboard users to bypass it efficiently. Vanilla CSS was necessary as no utility framework was present.
**Action:** When working on vanilla HTML/CSS projects, ensure a `.skip-link` class is added with absolute positioning off-screen until focused, providing seamless keyboard accessibility without disrupting visual design.
